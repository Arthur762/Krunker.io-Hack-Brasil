# Bem-vindo ao Krunker.io Hack's Brasil!
Este é um Hack Para Krunker.io carregado como uma extensão do Chrome.

##: muscle: recursos

* Aimbot
* Apelidos visíveis através das paredes
* Sem recuo
* Contador FPS

###: information_source: instruções

- Pressione a tecla Z para ativar / desativar o Hack.
- Se você precisar desabilitar temporariamente a mira automática, apenas segure a tecla Shift esquerda.

##: hammer: Instalação

1. Baixe [este repositório como um arquivo ZIP] (https://github.com/Arthur762/Krunker.io-Hack-Brasil/archive/master.zip).
2. Extraia o arquivo ZIP que você acabou de baixar.
3. Vá para "chrome: // extensions" no seu navegador. * Certifique-se de ter ativado o modo de desenvolvedor. *
4. Clique em "Carregar Sem Compactação" e selecione a subpasta chamada "ChromeExtension" que está Extraida (chamado `Krunker.io Hack's Brasil)
5. Abra o [Krunker.io] (http://krunker.io).
6. Divirta-se!
